Learning to use git and github

The 2 steps to push your work in a github repository

Modify your files locally

  1. Modify your files in the working directory
  2. Add the modified files in the stage phase by typing: git add "filename"
  3. Once your are ready to to change type: git commit -m "description of the change" to add the new version of the files in git repository

Push the modified files in Github

  1. Once the file in the git repo, it's time to push it to Github, in order to do it you'll have to create a repo in Github and take the URL
  2. Then type in the command line: git remote add origin "URL"
  3. Now that you have the address of the Github repo you can push your work in it
  4. by first positioning yourself in the repo main branch by typing : git branch -M main to connect yoursel to the main branch
  5. Finally type: git push -u origin main to push the modified files inside the Github repo
